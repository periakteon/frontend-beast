# HTML Element: `<cite>`

The `<cite>` element in HTML is used to mark up the title of a creative work, such as a book, movie, song, or other piece of content that you want to **reference** or **emphasize**.

## Syntax

Its syntax is quite straightforward:

```html showLineNumbers /cite/1,3
<cite>Text to be cited</cite>
```

## Attributes

The `<cite>` element has a few attributes that can be used to provide additional information:

### `title`

The `title` attribute can be used to provide additional information about the cited work. It is displayed as a tooltip when the user hovers over the element.

```html showLineNumbers /title/1
<cite title="The title of the cited work">Text to be cited</cite>
```

**Output:**

<cite title="The title of the cited work">Text to be cited</cite>

_(Hover over the element to see the tooltip)_

## Example

### Basic Usage

```html showLineNumbers /cite/
<p>
  According to <cite>Shakespeare's Sonnet 18</cite>, "Shall I compare thee to a
  summer's day?"
</p>
```

**Output:**

<p>
  According to <cite>Shakespeare's Sonnet 18</cite>, "Shall I compare thee to a
  summer's day?"
</p>

---

### With `title` Attribute

```html showLineNumbers /title/
<p>
  According to <cite title="Shakespeare's Sonnet 18">Sonnet 18</cite>, "Shall I
  compare thee to a summer's day?"
</p>
```

**Output:**

<p>
  According to <cite title="Shakespeare's Sonnet 18">Sonnet 18</cite>, "Shall I
  compare thee to a summer's day?"
</p>

_(Hover over the element to see the tooltip)_

## Best Practices

To ensure proper usage of the `<cite>` element, consider the following best practices:

- Use `<cite>` for the titles of creative works like books, movies, songs, etc.
- Include relevant information in the `title` attribute for additional context.
- Avoid using `<cite>` for generic or non-creative work titles.

## Advanced Features

While `<cite>` is primarily used for citing titles, you can get creative with it in more advanced scenarios:

### Advanced Usage 1: Citing Multiple Works

You can use multiple `<cite>` elements for citing multiple works in a single sentence.

```html showLineNumbers /cite/
<p>
  According to <cite>Shakespeare's Sonnet 18</cite> and
  <cite>Keats' Ode to a Nightingale</cite>,...
</p>
```

**Output:**

<p>
  According to <cite>Shakespeare's Sonnet 18</cite> and{" "}
  <cite>Keats' Ode to a Nightingale</cite>,...
</p>

---

### Advanced Usage 2: Styling

You can style the `<cite>` element using [**CSS**](/css) to match your design requirements.

```html showLineNumbers /cite/2,3
<style>
  cite {
    font-style: italic;
    color: #0077b6;
  }
</style>
<p>According to <cite>Shakespeare's Sonnet 18</cite>,...</p>
```

**Output:**

export const Styling = () => {
  return (
    <p>
      According to {""}
      <cite style={{ fontStyle: "italic", color: "#0077b6" }}>
        Shakespeare's Sonnet 18
      </cite>, ...
    </p>
  );
};

<Styling />

Remember, the `<cite>` element is a valuable tool for providing proper attribution and citation within your HTML content. Using it correctly can enhance the clarity and professionalism of your web pages.
