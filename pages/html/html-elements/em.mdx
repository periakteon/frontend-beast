# HTML Element: `<em>`

The `<em>` element, short for **emphasis**, is used to indicate text that should be emphasized in some way, typically by rendering it in _italic_.

## Syntax

It is an [**inline-level element**](https://developer.mozilla.org/en-US/docs/Glossary/Inline-level_content) and is written using the following syntax:

```html showLineNumbers /em/
<em>Emphasized Text</em>
```

**Output:**

<em>Emphasized Text</em>

## Attributes

The `<em>` element does not have any specific attributes of its own. However, it can accept [**global attributes**](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes) such as `class`, `id`, `style`, and more, which allow you to apply [**CSS**](/css) styles or [**JavaScript**](/javascript) interactions.

## Examples

### Example 1: Emphasizing Text

The `<em>` element is used to emphasize text. It is typically rendered in _italic_ by default, but this can be changed using [**CSS**](/css).

```html showLineNumbers /em/
<p>This is a <em>very</em> important word.</p>
```

### Example 2: Emphasizing Text in a Quote

The `<em>` element can be used to emphasize text in a quote.

```html showLineNumbers /em/

```

### Example 3: Nesting with Strong Emphasis

The `<em>` element can be nested inside the [`<strong>`](/html/html-elements/strong) element to create a stronger emphasis.

```html showLineNumbers /em/
<p>
  This is a <strong>very <em>very</em> important</strong> word.
</p>
```

## Best Practices

- Use the `<em>` element to provide semantic emphasis to text that deserves special attention.
- Avoid using `<em>` solely for styling purposes; instead, use [**CSS**](/css) for styling.
- Consider using `<strong>` for stronger emphasis when necessary.

## Advanced Features

- **Accessibility:** Ensure that emphasized text is accessible to screen readers and other assistive technologies by using appropriate [**ARIA roles**](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles) and labels.

- **Styling:** Customize the appearance of emphasized text using [**CSS**](/css) to match your design requirements.

- **JavaScript Interaction:** You can use JavaScript to interact with `<em>` elements, such as adding event listeners for specific actions when users interact with emphasized text.
