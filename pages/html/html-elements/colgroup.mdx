# HTML Element: `<colgroup>`

The `<colgroup>` element in HTML is used to group a set of `<col>` elements that represent column properties for an HTML table.

## Syntax

Here's the basic syntax of the `<colgroup>` element:

```html showLineNumbers /colgroup/
<colgroup>
  <!-- col elements -->
</colgroup>
```

The `<colgroup>` element acts as a **container** for one or more `<col>` elements.

## Attributes

### `span`

This attribute specifies the number of columns that the `<colgroup>` element should span. It determines how many columns the properties defined in the `<col>` elements within the group apply to.

```html showLineNumbers /span/
<table>
  <colgroup span="2">
    <col style="background-color: red;" />
    <col style="background-color: yellow;" />
  </colgroup>
  <tr>
    <th>First Name</th>
    <th>Last Name</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Doe</td>
  </tr>
</table>
```

## Best Practices

- Use `<colgroup>` when you need to apply common column properties to multiple columns within a table.
- Specify a width for columns if needed to control the layout.
- Use semantic HTML elements where possible instead of applying styles directly in `<col>` elements.

## Advanced Features

- **Responsive Tables:** You can use media queries in [**CSS**](/css) to change the `<col>` element attributes (e.g., `width`) based on the screen size, making your tables more responsive.

- **Styling Columns:** You can use [**CSS**](/css) to style columns differently within a `<colgroup>`. This is useful for highlighting specific columns or applying unique styles.

- **Accessibility:** Ensure that your tables and column groups are accessible by providing appropriate ARIA attributes and labels for screen readers.
