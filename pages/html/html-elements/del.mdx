# HTML Element: `<del>`

The `<del>` element is used to represent deleted or removed text within a document. It is often used in conjunction with the `<ins>` element to represent changes to a document.

## Syntax

Here's the basic syntax of the `<del>` element:

```html showLineNumbers /del/
<del>Deleted text</del>
```

**Output:**

<del>Deleted text</del>

If we want to use it with the `<ins>` element, we can do it like this:

```html showLineNumbers /ins/ {2}
<del>Deleted text</del> <ins>Inserted text</ins>
```

**Output:**

<del>Deleted text</del>
<ins>Inserted text</ins>

## Attributes

### `cite`

The `cite` attribute specifies the URL of the resource that explains the change. It is used to provide a reference to the source of the change.

```html showLineNumbers /cite/
<del cite="https://www.example.com">Deleted text</del>
```

**Output:**

<del cite="https://www.example.com">Deleted text</del>

### `datetime`

The `datetime` attribute specifies the date and time when the change was made. It is used to provide a reference to the time of the change.

```html showLineNumbers /datetime/
<del datetime="2023-09-02">Deleted text</del>
```

**Output:**

<del datetime="2023-09-02">Deleted text</del>

## Best Practices

- Use the `<del>` element to indicate content that has been removed or deleted, providing a clearer understanding of the document's revision history.
- Always use the `datetime` attribute to specify when the deletion occurred for improved context.
- Provide a meaningful value for the `cite` attribute if possible, linking to the original source where the deletion took place.

## Advanced Features

Advanced usage of the `<del>` element may involve scripting or dynamic modifications of the document based on user interactions or server data. For example, you can use JavaScript to apply the `<del>` element to text that has been removed in a collaborative editing environment.
