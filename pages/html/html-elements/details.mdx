# HTML Element: `<details>`

The `<details>` element in HTML is used to create a disclosure widget that can be toggled open or closed to reveal or hide additional content.

## Syntax

Here's the basic syntax of the `<details>` element:

```html showLineNumbers /details/
<details>
  <summary>Summary Text</summary>
  <!-- Content to be revealed or hidden goes here -->
</details>
```

**Output:**

<details>
  <summary>Summary Text</summary>
  Content to be revealed or hidden goes here
</details>

- **`<details>`:** This is the container element that wraps the disclosure widget.
- **`<summary>`:** Inside the `<details>` element, you place a `<summary>` element that provides a brief title or label for the disclosure widget.
- **Content:** Any content placed within the `<details>` element will be revealed or hidden when the widget is toggled.

## Attributes

The `<details>` element supports the following attributes:

### `open`

The `open` attribute is a boolean attribute that, when present, indicates that the disclosure widget should be open by default.

```html showLineNumbers /open/
<details open>
  <summary>Summary Text</summary>
  Content to be revealed or hidden goes here
</details>
```

**Output:**

<details open>
  <summary>Summary Text</summary>
  Content to be revealed or hidden goes here
</details>

## Best Practices

- Use `<summary>` to provide a clear and concise label for the disclosure widget.
- Ensure that the content inside the `<details>` element is relevant to the summary.
- Provide an alternative display for browsers that don't support the `<details>` element by placing essential content outside the element.

## Advanced Features

### Nesting

You can nest `<details>` elements inside each other to create a hierarchical disclosure widget.

```html showLineNumbers
<details>
  <summary>Summary Text</summary>
  Content to be revealed or hidden goes here
  <details>
    <summary>Summary Text</summary>
    Content to be revealed or hidden goes here
  </details>
</details>
```

**Output:**

<details>
  <summary>Summary Text</summary>
  Content to be revealed or hidden goes here
  <details>
    <summary>Summary Text</summary>
    Content to be revealed or hidden goes here
  </details>
</details>
