# HTML Element: `<embed>`

The `<embed>` element in HTML is used to embed external content or multimedia, such as audio, video, or interactive applications, within an HTML document

## Syntax

Here's the basic syntax of the `<embed>` element:

```html showLineNumbers
<embed src="URL" type="media_type" [optional attributes] />
```

The `<embed>` element is an empty element, meaning it doesn't have a closing tag. It must have a `src` attribute, which specifies the URL of the resource to be embedded. It also has a `type` attribute, which specifies the media type of the resource.

## Attributes

The `<embed>` element has the following attributes:

### `src`

The `src` attribute specifies the URL of the resource to be embedded. It can be an absolute or relative URL.

### `type`

The `type` attribute specifies the media type of the resource. It can be any of the **MIME types** listed [here](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types/Common_types).

### `width`

The `width` attribute specifies the width of the embedded resource in pixels. It can be any positive integer.

### `height`

The `height` attribute specifies the height of the embedded resource in pixels. It can be any positive integer.

## Example

### Video Embed

Here's an example of embedding a video using the `<embed>` element:

```html showLineNumbers /video/
<embed
  src="https://www.youtube.com/embed/9bZkp7q19f0"
  type="video/mp4"
  width="560"
  height="315"
/>
```

**Output:**

## Best Practices

- Always specify the `src` and `type` attributes to ensure compatibility.
- Use the `width` and `height` attributes to define the dimensions of the embedded content.
- Provide alternative content within `<object>` or `<iframe>` elements for browsers that don't support `<embed>`.
- Be cautious with `autoplay`, as it can be intrusive to users.

## Advanced Features

### `autoplay`

The `autoplay` attribute specifies that the embedded content should automatically start playing as soon as it's ready. It's a boolean attribute, meaning it doesn't require a value.

```html showLineNumbers /autoplay/
<embed
  src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3"
  type="audio/mp3"
  width="350"
  height="75"
  autoplay
/>
```

### `loop`

The `loop` attribute specifies that the embedded content should start over again when it reaches the end. It's a boolean attribute, meaning it doesn't require a value.

```html showLineNumbers /loop/
<embed
  src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3"
  type="audio/mp3"
  width="350"
  height="75"
  loop
/>
```
