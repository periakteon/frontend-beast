# HTML Element: `<code>`

The `<code>` element in HTML is used to define a piece of computer code. It is typically used for displaying code snippets within a web page's content.

## Syntax

Here's the basic syntax of the `<code>` element:

```html showLineNumbers /code/1,3
<code> Your code here </code>
```

You simply enclose your code within the opening `<code>` tag and the closing `</code>` tag.

## Attributes

The `<code>` element doesn't have many specific attributes of its own. However, it can accept global attributes such as:

- `class`: To apply [**CSS**](/css) styles or JavaScript functions to the code block.
- `id`: To uniquely identify the code block.
- `style`: For inline [**CSS**](/css) styling.
- `title`: To provide additional information about the code block (tooltip).
- `data-*`: Custom data attributes for storing extra information.

Here's an example of how to use some of these attributes:

```html showLineNumbers {2-5}
<code
  class="highlighted"
  id="myCode"
  style="font-size: 14px;"
  title="Important code"
>
  // Your code here
</code>
```

## Best Practices

When using the `<code>` element, consider the following best practices:

- **Use it sparingly:** Only use `<code>` for actual code snippets; don't use it for plain text that resembles code.
- **Escape HTML entities:** If you want to display code containing HTML tags, use HTML entities (`&lt;` and `&gt;`) to prevent rendering.
- **Provide context:** Surround your `<code>` element with appropriate text or headings to give context to the code.

## Advanced Features

The `<code>` element is often used in combination with other elements like `<pre>` for displaying code with whitespace preservation, and `<span>` for inline code highlighting. Here's an advanced example:

```html showLineNumbers /code/
<pre>
   <code class="python">
      def factorial(n):
          if n == 0:
              return 1
          else:
              return n * factorial(n-1)
   </code>
</pre>
```

**Output:**
