# HTML Element: `<col>`

The `<col>` element in HTML is used within a [**`<colgroup>`**](/html/html-elements/colgroup) element to define properties for columns in an HTML table.

## Syntax

Here's the basic syntax of the `<col>` element:

```html showLineNumbers /col/
<col />
```

The `<col>` element is an empty element, meaning it doesn't have a closing tag.

## Attributes

The `<col>` element supports the following attributes:

### `span`

The `span` attribute specifies the number of columns the `<col>` element should span. The default value is `1`.

```html showLineNumbers /span/""
<col span="2" />
```

## Example

Here's an example of the `<col>` element in action:

```html showLineNumbers /col/2,4
<table>
  <colgroup>
    <col span="2" style="background-color: red;" />
    <col style="background-color: yellow;" />
  </colgroup>
  <tr>
    <th>First Name</th>
    <th>Last Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Doe</td>
    <td>28</td>
  </tr>
</table>
```

## Best Practices

- Use `<col>` elements within a `<colgroup>` for better organization and control of column attributes.
- Avoid using the width attribute and prefer [**CSS**](/css) for styling to ensure better responsiveness.
- Keep the use of `<col>` elements minimal for simpler tables, as they are more beneficial for larger tables with complex layouts.

## Advanced Features

- Combining `<col>` elements with [**CSS**](/css) classes for fine-grained control over styling.
- Using JavaScript to dynamically modify `<col>` attributes based on user interactions.
- Employing media queries to adjust column properties based on screen size for responsive tables.

These advanced usages can help you create more dynamic and responsive tables using the `<col>` element in HTML.
