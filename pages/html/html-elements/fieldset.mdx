# HTML Element: `<fieldset>`

The `<fieldset>` element is used to **group** related **form** elements within a web form.

## Syntax

Here's the basic syntax of the `<fieldset>` element:

```html showLineNumbers /fieldset/
<form>
  <fieldset>
    <!-- form elements -->
  </fieldset>
</form>
```

The `<fieldset>` element contains other form-related elements, typically `<legend>` for a **title**, and **form controls** like `<input>`, `<textarea>`, and `<select>`.

## Attributes

### `disabled`

The `disabled` attribute is used to disable all the form controls within the `<fieldset>` element.

```html showLineNumbers /disabled/
<form>
  <fieldset disabled>
    <!-- form elements -->
  </fieldset>
</form>
```

### `form`

Specifies which `<form>` element the `<fieldset>` belongs to, using the form's `id` attribute.

```html showLineNumbers /form/1,3,6 /id/
<form id="form">
  <fieldset form="form">
    <!-- form elements -->
  </fieldset>
</form>
```

### `name`

Assigns a name to the `<fieldset>`, which can be useful for scripting.

```html showLineNumbers /name/
<form>
  <fieldset name="fieldset">
    <!-- form elements -->
  </fieldset>
</form>
```

## Example

### Basic Example

```html showLineNumbers
<form>
  <fieldset>
    <legend>Contact Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" /><br /><br />

    <input type="email" id="email" name="email" /><br /><br />

    <label for="city">City:</label>
    <input type="text" id="city" name="city" /><br /><br />
  </fieldset>
</form>
```

export const FieldSetExample = () => {

## Best Practices

- Always include a `<legend>` element to provide a meaningful description for the `<fieldset>`.
- Use `<label>` elements with their for attribute pointing to the respective form controls for better accessibility.
- Avoid nesting `<fieldset>` elements as it can "lead to confusion in screen readers.

## Advanced Features

### Styling `<fieldset>` Elements

You can apply [**CSS**](/css) to style `<fieldset>` elements, including borders, padding, and background colors.

```css showLineNumbers
fieldset {
  border: 1px solid #ccc;
  padding: 1rem;
  background-color: #f5f5f5;
}
```

### Styling `<legend>` Elements

You can apply [**CSS**](/css) to style `<legend>` elements, including font size, font weight, and text color.

```css showLineNumbers
legend {
  font-size: 1.5rem;
  font-weight: bold;
  color: #333;
}
```

### Styling `<label>` Elements

You can apply [**CSS**](/css) to style `<label>` elements, including font weight, margin, and text color.

```css showLineNumbers
label {
  font-weight: bold;
  margin-bottom: 0.5rem;
  color: #333;
}
```

### Styling `<input>` Elements

You can apply [**CSS**](/css) to style `<input>` elements, including padding, margin, border radius, and border.

```css showLineNumbers
input {
  padding: 0.5rem;
  margin-bottom: 1rem;
  border-radius: 0.3rem;
  border: 1px solid #ccc;
}
```

### JavaScript Events

You can use JavaScript to dynamically enable or disable `<fieldset>` elements based on user actions.

```js showLineNumbers
const fieldset = document.querySelector("fieldset");
const checkbox = document.querySelector("input[type=checkbox]");
checkbox.addEventListener("change", () => {
  fieldset.disabled = !fieldset.disabled;
});
```

**Output:**

export const JavaScriptFieldSet = () => {
    const [isMounted, setIsMounted] = React.useState(false);
    const [isDisabled, setIsDisabled] = React.useState(false);

    React.useEffect(() => {
        setIsMounted(true);
    }, []);

    if (!isMounted) {
        return null;
    }

    const handleCheckboxChange = () => {
    setIsDisabled(!isDisabled);
    };

    return (
    <div>
      <fieldset style={{ opacity: isDisabled ? 0.5 : 1 }}>
        <legend>Fieldset</legend>
        <label>
          <input type="checkbox" onChange={handleCheckboxChange} style={{ marginRight: '0.5rem' }} />
          Disable Fieldset
        </label>
        <p>This is some content inside the fieldset.</p>
      </fieldset>
    </div>

);
}

### Validation

Combine with HTML5 form validation attributes like required or custom validation logic to validate form controls within a `<fieldset>`.

```html showLineNumbers
<fieldset>
  <legend>Payment Options</legend>
  <input type="radio" id="credit" name="payment" value="credit" required />
  <label for="credit">Credit Card</label>
  <input type="radio" id="paypal" name="payment" value="paypal" />
  <label for="paypal">PayPal</label>
</fieldset>
```
