# JavaScript: How to Use?

JavaScript is a versatile and powerful programming language that allows you to add interactivity and dynamic behavior to your web pages. In this comprehensive guide, we will explore how to use JavaScript in [**HTML**](/html) files, covering various techniques and providing plenty of examples.

## Introduction

JavaScript is a client-side (in **Node.js**, server-side) scripting language that enhances the functionality of web pages by enabling interactions, data manipulation, and more. To get started, let's look at how to include JavaScript in your [**HTML**](/html) documents.

## Adding JavaScript to HTML

There are multiple ways to include JavaScript in your [**HTML**](/html) files:

### Inline JavaScript

You can embed JavaScript directly within your HTML file using the `<script>` tag. Here's an example:

```html showLineNumbers /script/ {8-11} filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>Inline JavaScript Example</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <script>
      // JavaScript code here
      alert("This is an inline JavaScript example!");
    </script>
  </body>
</html>
```

### External JavaScript File

To keep your code organized and reusable, it's common to place JavaScript code in external files with a `.js` extension. Here's how you can link an external JavaScript file to your HTML:

```html showLineNumbers /script.js/ {5} filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>External JavaScript Example</title>
    <script src="script.js"></script>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
```

In this example, the JavaScript code is stored in a file named `script.js`.

<Callout type="info" emoji="ℹ️">
  Put the `script.js` and `index.html` files in the **same directory**.
  Otherwise, you'll need to specify the path to the JavaScript file.
</Callout>

## JavaScript Placement

The placement of JavaScript code within your HTML file can affect how it interacts with the HTML elements. Here are some key points to consider:

### Placing JavaScript in the `<head>` section

When you place JavaScript in the `<head>` section of your HTML file, it's executed **before the HTML content is fully loaded**. This can be useful for defining global variables or functions that will be used throughout the page.

```html showLineNumbers {3-10} /head/1,4 filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript in Head</title>
    <script>
      // JavaScript code in the head
      let message = "Hello from the head!";
      alert(message);
    </script>
  </head>
  <body>
    <!-- HTML content -->
  </body>
</html>
```

<Callout type="warning" emoji="⚠️">
 If you place JavaScript in the `<head>` section, make sure to use the `defer` attribute to ensure that the code is executed after the HTML content is loaded. Otherwise, you may encounter errors.
 Look at the [third section](#3-placing-javascript-in-the-head-section-with-defer) for more details.
</Callout>

### 2. Placing JavaScript at the end of `<body>`

Placing JavaScript at the end of the `<body>` tag ensures that it runs **after the HTML content has loaded**. This is often **recommended** to improve page loading performance.

```html showLineNumbers {6-13} /body/1,4 filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript at the End of Body</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <script>
      // JavaScript code at the end of body
      let message = "Hello from the end of body!";
      alert(message);
    </script>
  </body>
</html>
```

### 3. Placing JavaScript in the `<head>` section with `defer`

You can also place JavaScript in the `<head>` section and use the `defer` attribute to ensure that it runs **after the HTML content has loaded**. This is a good option if you want to keep your JavaScript code in the `<head>` section.

```html showLineNumbers {3-10} /head/1,4 /defer/1 filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript in Head with Defer</title>
    <script defer>
      // JavaScript code in the head with defer
      let message = "Hello from the head with defer!";
      alert(message);
    </script>
  </head>
  <body>
    <!-- HTML content -->
  </body>
</html>
```


## Handling Events

JavaScript enables you to respond to various events, such as **clicks**, **keypresses**, and **form submissions**. Here's an example of handling a click event:


## Asynchronous JavaScript

JavaScript can perform **asynchronous** operations, such as making **API requests** or **handling timers**. Here's a simple example using `setInterval`:

```html showLineNumbers {7-28} filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>Asynchronous JavaScript</title>
  </head>
  <body>
    <p id="demo">This message will change after 3 seconds.</p>
    <button id="updateButton">Update Message</button>
    <p id="countdown">Countdown: <span id="secondsLeft">3</span> seconds</p>
    <script>
      document
        .getElementById("updateButton")
        .addEventListener("click", function () {
          let seconds = 3;

          const countdownDisplay = document.getElementById("secondsLeft");

          const countdownInterval = setInterval(function () {
            seconds--;
            countdownDisplay.textContent = seconds;

            if (seconds === 0) {
              clearInterval(countdownInterval);
              document.getElementById("demo").innerHTML = "Updated message!";
            }
          }, 1000);
        });
    </script>
  </body>
</html>
```

**Output:**

export const AsynchronousJavaScriptComponent = () => {
    const [isMounted, setIsMounted] = React.useState(false);
    const [text, setText] = React.useState("This message will change after 3 seconds.");
    const [secondsLeft, setSecondsLeft] = React.useState(3);

    React.useEffect(() => {
        setIsMounted(true);
    }, []);

    if (!isMounted) {
        return null;
    }

    const handleClick = () => {
        let seconds = 3;

        const countdownInterval = setInterval(function() {
            seconds--;
            setSecondsLeft(seconds);

            if (seconds === 0) {
                clearInterval(countdownInterval);
                setText("Updated message!");
            }
        }, 1000);
    };

    return (
        <div style={{ padding: "1rem", backgroundColor: "#f7fafc", borderRadius: "0.5rem" }}>
            <p id="demo">{text}</p>
            <button
            style={{ padding: "1rem 2rem", fontSize: "1rem", backgroundColor: "#1a202c", color: "#fff", borderRadius: "0.5rem", hover: { backgroundColor: "#2d3748" } }}
                onClick={handleClick}
            >
                Update Message
            </button>
            <p id="countdown">Countdown: <span id="secondsLeft">{secondsLeft}</span> seconds</p>
        </div>
    );

}
