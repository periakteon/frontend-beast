# CSS Property: `accent-color`

The `accent-color` property in CSS is used to define **the accent color of an element**. Accent colors are typically used for **highlighting** or **emphasizing** certain parts of a webpage, such as links or buttons. This property allows web developers to customize the accent color to match the overall design of their website.

## Syntax

The syntax for using the `accent-color` property is as follows:

```css showLineNumbers
selector {
  accent-color: <color>;
}
```

Where:

- `selector` is the [**HTML element**](/html/html-elements) or class to which you want to apply the accent color.
- `value` is the color value you want to set as the accent color.

The `accent-color` property is a **custom property** that can be used to define the accent color of an element. The value of this property can be any valid CSS color value, such as a **hexadecimal color code**, **RGB color value**, or **color keyword**.

## Property Values

The `accent-color` property accepts various types of values:

### Color Values

You can specify the accent color using color values such as **hexadecimal**, **RGB**, **RGBA**, **HSL**, **HSLA**, or **color names**.

#### **Hexadecimal Color Code**

The hexadecimal color code is a six-digit code that represents the amount of red, green, and blue in a color. For example, `#ff0000` is the hexadecimal color code for red.

```css showLineNumbers filename="style.css"
p {
  accent-color: #ff0000;
}
```

---

#### **RGB Color Value**

The RGB color value is a three-digit code that represents the amount of red, green, and blue in a color. For example, `rgb(255, 0, 0)` is the RGB color value for red.

```css showLineNumbers filename="style.css"
p {
  accent-color: rgb(255, 0, 0);
}
```

---

#### **RGBA Color Value**

The RGBA color value is a four-digit code that represents the amount of red, green, blue, and alpha (transparency) in a color. For example, `rgba(255, 0, 0, 0.5)` is the RGBA color value for red with 50% transparency.

```css showLineNumbers filename="style.css"
p {
  accent-color: rgba(255, 0, 0, 0.5);
}
```

---

#### **HSL Color Value**

The HSL color value is a three-digit code that represents the hue, saturation, and lightness of a color. For example, `hsl(0, 100%, 50%)` is the HSL color value for red.

```css showLineNumbers filename="style.css"
p {
  accent-color: hsl(0, 100%, 50%);
}
```

---

#### **HSLA Color Value**

The HSLA color value is a four-digit code that represents the hue, saturation, lightness, and alpha (transparency) of a color. For example, `hsla(0, 100%, 50%, 0.5)` is the HSLA color value for red with 50% transparency.

```css showLineNumbers filename="style.css"
p {
  accent-color: hsla(0, 100%, 50%, 0.5);
}
```

---

#### **Color Keyword**

The color keyword is a predefined color name that represents a specific color. For example, `red` is the color keyword for red.

```css showLineNumbers filename="style.css"
p {
  accent-color: red;
}
```

## Example

```css showLineNumbers
/* Using a hexadecimal color value */
.button {
  accent-color: #ff5733;
}

/* Using an RGB color value */
.link {
  accent-color: rgb(0, 128, 255);
}

/* Using a color name */
.alert {
  accent-color: red;
}
```

## Advanced Features

### Using the `accent-color` Property with `:focus-visible` Pseudo-class

The `:focus-visible` pseudo-class is used to apply styles to an element when it is focused using the keyboard or a similar input device. This pseudo-class can be used in conjunction with the `accent-color` property to apply the accent color to an element when it is focused.

```css showLineNumbers
/* Using the :focus-visible pseudo-class */
.button:focus-visible {
  accent-color: #ff5733;
}
```

### Using the `accent-color` Property with `:hover` Pseudo-class

The `:hover` pseudo-class is used to apply styles to an element when the user hovers over it with their mouse cursor. This pseudo-class can be used in conjunction with the `accent-color` property to apply the accent color to an element when it is hovered over.

```css showLineNumbers
/* Using the :hover pseudo-class */
.button:hover {
  accent-color: #ff5733;
}
```

### With Custom Variables

The `accent-color` property can be used with custom variables to define the accent color of an element. This allows web developers to easily change the accent color of an element without having to modify the CSS code.

```css showLineNumbers
/* Using custom variables */
:root {
  --accent-color: #ff5733;
}

.button {
  accent-color: var(--accent-color);
}
```
